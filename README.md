## Install

```sh
$ git clone https://github.com/SeechRobin/crypto.git
$ npm install
$ npm start
```

## Proxy

I used Cors-Anywhere as the CORS Proxy server. You have to run this as well.

```sh
$ git clone https://github.com/Rob--W/cors-anywhere.git
$ cd cors-anywhere/
$ npm install
$ npm start
```
